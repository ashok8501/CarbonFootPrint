# 🌍 Advanced Carbon Footprint Calculator

A modern, interactive web application built using Streamlit to estimate and analyze personal carbon emissions based on daily lifestyle choices.

---

## 🚀 Features

* 🚗 Multi-mode transport tracking (car, bike, bus, train)
* ⚡ Appliance-based electricity calculation
* 🍽 Detailed food analysis (veg, chicken, mutton, dairy)
* 🗑 Waste generation & recycling impact
* ✈ Flight emission tracking
* 📊 Interactive emission breakdown charts
* 🌍 Global comparison (India vs World averages)
* 🎯 Sustainability scoring system
* 💡 Personalized eco-friendly suggestions
* 🌳 Carbon offset estimation (trees required)

---

## 🧠 How It Works

The calculator uses approximate emission factors derived from environmental research to estimate CO₂ emissions across different lifestyle categories:

* Transport emissions based on distance and mode
* Electricity usage based on appliance consumption
* Food impact based on dietary habits
* Waste impact including recycling behavior
* Air travel emissions per year

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/carbon-footprint-calculator.git
cd carbon-footprint-calculator
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
